# learn-wasm

```bash
# Initialize the project
# https://github.com/lencx/create-mpl

# npm 6.x
npm init mpl@latest my-app --type web

# npm 7+, extra double-dash is needed:
npm init mpl@latest my-app -- --type web
```

## Pre-Install

```bash
cargo install rsw wasm-pack
```

## Start

```bash
yarn dev
```
